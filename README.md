# Upgrade Portal

This project is a web-based portal for managing:

- EMR Upgrade Scheduling  
- Shell Requests  

## Technologies Used

- ASP.NET Core MVC  
- C#  
- SQL Server  
- Entity Framework Core  

## Features

- Dashboard  
- Schedules (Create, Sort, Complete, Cancel)  
- Shell Requests (Full Form + Sorting)
