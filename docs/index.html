Upgrade Portal System

About the Project

Project Overview

Developed as a University of the People MSIT Capstone Project by Abdul Rauf Ibrahimkhail, May 2026.

⚠
The Problem
Healthcare IT teams tracked upgrade schedules in Excel and submitted environment requests through Jira — with no field validation, no status visibility, and repeated clarification cycles between Professional Services and TechOps.
✓
The Solution
A purpose-built ASP.NET Core MVC portal that enforces complete request intake through mandatory form validation, automates TechOps email notifications via SendGrid, and gives administrators real-time status visibility.
🎯
Outcome
10/10 test cases passed across integration, system, and acceptance testing levels. Three defects identified and resolved. Login response 1.2s · CSV export 2.4s · Zero build errors.
Features

System Modules

Six integrated modules covering the full operational workflow.

01
Authentication & 2FA
SHA-256 password hashing · Two-factor authentication via SendGrid · Session management · Password reset workflow · Role-based access control
02
Schedule Management
Create and manage EMR upgrade schedules · Status tracking (Pending / In Progress / Completed / Cancelled) · CSV export · Full audit trail
03
Shell Requests
Structured new environment intake form · Mandatory field validation · Automatic TechOps email notification on submission via SendGrid
04
User Administration
Create users with hashed passwords · Role assignment (Admin / User) · Per-user 2FA flag · Active / inactive account control
05
Reporting & CSV Export
Server-side CSV generation using StringBuilder · EF Core query · UTF-8 FileContentResult · Browser download — no client-side JavaScript required
06
Audit Logging
Serilog structured logging · Daily rolling files in /logs · Captures login events, DB writes, email dispatch status codes, and application errors
System Design

Architecture

Three-tier layered architecture with external SendGrid integration.

Presentation Tier
Razor Views (.cshtml)

Login · Dashboard · Schedules · Shell Requests · Reports · User Management
Application Tier
ASP.NET Core MVC

Controllers · Service classes · Middleware · Input validation · Access control
Data Tier
SQL Server + EF Core

Users · Roles · UpgradeSchedules · ShellRequests · AuditLog
External Integration
SendGrid Web API

2FA codes · Password resets · TechOps shell request notifications
