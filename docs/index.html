About the Project

Project Overview

Developed by Abdul Rauf Ibrahimkhail as a University of the People MSIT Capstone Project, May 2026.

⚠
The Problem
Healthcare IT teams tracked upgrade schedules in Excel and submitted environment requests through Jira with no field validation, no status visibility, and repeated clarification cycles that delayed client go-live dates.
✓
The Solution
A purpose-built ASP.NET Core MVC portal enforcing complete request intake through mandatory validation, automating TechOps email notifications via SendGrid, and giving administrators real-time status visibility.
🎯
Outcome
10/10 test cases passed across integration, system, and acceptance levels. Three defects identified and resolved. Login response 1.2s · CSV export 2.4s · Zero build errors.
Features

System Modules

Six integrated modules covering the full operational workflow.

01
Authentication & 2FA
SHA-256 hashing · 2FA via SendGrid · Session management · Password reset · Role-based access control at controller level
02
Schedule Management
Create and manage EMR upgrade records · Status — Pending / In Progress / Completed / Cancelled · CSV export · Serilog audit trail
03
Shell Requests
Structured environment intake form · Mandatory field validation · Automatic TechOps email on submit — no manual forwarding step
04
User Administration
Create users with hashed passwords · Role assignment — Admin / User · Per-user 2FA flag · Active/inactive control
05
Reporting & CSV Export
Server-side CSV via StringBuilder · EF Core query · UTF-8 FileContentResult · Browser download — no JavaScript required
06
Audit Logging
Serilog daily rolling files · Login events · DB writes · Email dispatch status · Application errors — full traceability
System Design

Architecture

Three-tier layered architecture with an external SendGrid integration layer — each tier communicates through well-defined contracts.

Tier 1 — Presentation
Razor Views (.cshtml)

Login · Dashboard · Schedules · Shell Requests · Reports · User Management
Tier 2 — Application
ASP.NET Core MVC

Controllers · Services · Middleware · Input validation · Session & access control
Tier 3 — Data
SQL Server + EF Core

Users · Roles · UpgradeSchedules · ShellRequests · AuditLog · Migrations
External — SendGrid
SendGrid Web API

2FA codes · Password resets · TechOps shell request alerts
